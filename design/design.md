# AIName 设计规范文档 (Design System)

本文档基于最新的 UI 设计迭代（参考 `design` 目录下的设计图），为 AIName 项目提供全局一致的视觉标准。后续所有新页面开发必须严格遵守此规范。

## 1. 配色规范 (Color Palette)

- **主色 (Primary)**: `#2563EB` (活力蓝) - 用于主按钮、激活状态、重要链接。
- **辅助色 (Secondary/Accent)**: `#DBEAFE` (浅蓝) - 用于标签背景、模块次要强调。
- **背景色 (Background)**:
  - 页面大背景: `#F9FAFB` 或 `#F3F4F6` (浅灰，提供呼吸感)。
  - 卡片/面板背景: `#FFFFFF` (纯白，突出版面)。
- **文字颜色 (Typography Color)**:
  - 主要标题/高亮正文: `#111827` (深灰/近黑，高对比度)。
  - 常规正文/标签: `#4B5563` 或 `#6B7280`。
  - 弱化说明/次要文字: `#9CA3AF`。
- **状态颜色 (Status)**:
  - 成功 (Success): `#10B981`，背景 `#D1FAE5`。
  - 失败/警告 (Error/Warning): `#EF4444`，背景 `#FEE2E2`。
- **边框颜色 (Border)**: `#E5E7EB` 或 `#D1D5DB`。

## 2. 字体规范 (Typography)

采用现代无衬线字体体系 (System-ui, Inter, Roboto, PingFang SC)。

- **页面大标题 (H1)**: 字体大小 `32px` / `36px`，字重 `Bold (700)`，行高 `1.4`。
- **模块标题/卡片标题 (H2)**: 字体大小 `20px` / `24px`，字重 `Semi-Bold (600)`，行高 `1.5`。
- **正文 (Body)**: 字体大小 `14px` / `16px`，字重 `Regular (400)`，行高 `1.6`。
- **小字说明 (Caption/Small)**: 字体大小 `12px`，字重 `Regular (400)`，行高 `1.5`。

## 3. 间距与留白 (Spacing)

以 `4px`/`8px` 为基础栅格单位。

- **页面内边距 (Page Padding)**: 移动端/小程序 `24px` (`30rpx`~`48rpx`)，桌面端 `32px` 或更大。
- **模块间距 (Section Gap)**: `24px` 或 `32px`，保证页面模块之间的呼吸感。
- **卡片内边距 (Card Padding)**: `24px` (`30rpx`)，内容宽松排版。
- **表单间距 (Form Item Gap)**: `16px`。
- **元素间距 (Element Gap)**: `8px` 或 `12px` (如按钮与按钮之间，文字与标签之间)。

## 4. 组件样式规范 (Component Styles)

- **按钮 (Buttons)**:
  - **主按钮**: 背景 `#2563EB`，文字 `#FFFFFF`，无边框，圆角 `8px` (或 `12rpx`)，字重 `500`，悬浮时可轻微改变透明度或加深颜色。部分大按钮可使用 `50rpx` 圆角。
  - **次按钮/描边按钮**: 背景 `#FFFFFF`，文字 `#4B5563`，边框 `1px solid #D1D5DB`，圆角 `8px`。
- **卡片 (Cards)**:
  - 背景色: `#FFFFFF`。
  - 边框: 通常无边框，或极为轻柔的 `1px solid #E5E7EB`。
  - 圆角 (Border Radius): `12px` 或 `16px` (大圆角营造现代、亲切感)。
  - 阴影 (Box Shadow): 柔软且克制的阴影，推荐使用 `0 4px 12px rgba(0,0,0,0.05)`，或者更轻量的弥散阴影。
- **输入框 (Inputs/Textareas)**:
  - 背景色: `#FFFFFF`。
  - 边框: `1px solid #D1D5DB`，聚焦时变为 `1px solid #2563EB`。
  - 圆角: `8px` (`12rpx`)。
  - 内边距: `12px 16px`。
- **导航栏/顶部状态栏 (Navigation/Header)**:
  - 背景色: `#FFFFFF`。
  - 阴影: `0 1px 3px rgba(0,0,0,0.05)`。
  - 圆角: 如果是悬浮式导航则带圆角，否则底部贴合无圆角。
- **标签 (Tags)**:
  - 成功标签: 背景 `#E8F5E9`，文字 `#4CAF50`，圆角 `30rpx`。
  - 失败标签: 背景 `#FFEBEE`，文字 `#F44336`，圆角 `30rpx`。

## 5. 整体布局原则 (Layout Principles)

- **白板/画布概念**: 页面底色使用浅灰 (`#F5F7FA`)，内容区使用白色卡片 (`#FFFFFF`) 承载，形成深度感。
- **清晰层级**: 通过字号和字重形成明确的视觉焦点。最重要的操作按钮必须是最吸引眼球的元素。
- **对齐**: 保持严格的左对齐或居中对齐，杜绝错落的边缘。
